# 💰 Калькулятор зарплати - Програма-баланс

> **Справедливі розрахунки заробітної плати для України** 🇺🇦

Сучасний SPA веб-додаток для розрахунку та прогнозування заробітної плати з підтримкою різних валют (через NBU API), систем оподаткування та можливостями кешування даних.

## 🚀 Основні функції (Оновлено)

### 💱 Підтримка валют (Актуальні курси НБУ)
- **₴ Гривня (UAH)** - національна валюта України
- **€ Євро (EUR)** - інтегрований автоматичний збір курсів з NBU API.
- **$ Долар США (USD)** - інтегрований автоматичний збір курсів з NBU API.
*Курси кешуються в `localStorage` на 24 години для роботи офлайн та швидкого доступу.*

### ⚖️ Розрахунок податкових систем (Нове!)
Ви можете обрати, як саме рахувати вашу зарплату. Додаток автоматично розрахує ваші обсяги "чистими на руки" (Net Income):
- **Без податків (Gross)** – сирі дані без вирахувань.
- **ФОП 3 група (5% + ЄСВ)** – автоматичне вирахування 5% податку та актуального розміру єдиного соціального внеску (ЄСВ) з прив'язкою до обраної валюти.
- **Офіційне працевлаштування (ПДФО 18% + ВЗ 1.5%)** – точні відрахування (19.5%) як прийнято за трудовим договором.

### 💾 Розумне збереження прогресу
- Всі ваші введені дані **автоматично зберігаються** у вашому браузері (`localStorage`).
- При закритті або оновленні вкладки ви не втратите свої цифри! Ваші налаштування автоматично відновляться.

### 📊 Категорії зарплат
- **Мінімальна зарплата** (6,000 - 12,000 ₴)
- **Середня зарплата** (15,000 - 35,000 ₴)
- **Висока зарплата** (40,000 - 100,000 ₴)

### 📈 Розрахунки та прогнози
- Річний ріст зарплати у відсотках
- Прогноз на 1-20 років вперед
- Розрахунок премій та бонусів
- **Інтерактивна візуалізація даних у графіку (Chart.js)**
- Детальна таблиця прогнозів (по роках та з доходами Net/Gross)

## 🛠 Технології та Архітектура
Проєкт недавно пройшов масштабний архітектурний рефакторинг та повністю базується на **Vanilla JavaScript (ES6 Класи)**.

- **Frontend**: HTML5, CSS3, JavaScript
- **State Management**: Вбудований менеджер на основі `localStorage`.
- **API**: Інтеграція з відкритим API Національного банку України.
- **Стилізація**: Скляний морфізм, градієнти, CSS-анімації.
- **Графіки**: `Chart.js` (відмальовка графіку росту доходів).
- **Іконки та Шрифти**: `Font Awesome` та `Inter (Google Fonts)`.

## 🎯 Як використовувати

### 1. Відкрийте додаток
Проєкт не потребує заднього бекенду або Node-сервера. Ви можете просто відкрити файл `index.html` у будь-якому веб-браузері.

### 2. Заповніть форму
- **Валюта**: Оберіть потрібну валюту (курси завантажаться автоматично)
- **Категорія зарплати**: Виберіть відповідний рівень
- **Оподаткування**: Виберіть метод оподаткування для зрізу "на руки".
- **Поточна зарплата** (Gross): Введіть вашу поточну зарплату до податків
- **Років досвіду** та **Річний ріст** (%).
- **Прогноз**: Виберіть кількість років.

### 3. Гарячі клавіші
- `Ctrl + Enter` - швидкий розрахунок та відображення звіту.
- `Tab` - зручна навігація формою.

### 4. Експорт даних
Ви можете зберегти вашу детальну таблицю прогнозів як звичайний `.csv` файл для подальшого аналізу в Excel / Google Sheets, або роздрукувати звіт прямо з браузера.
